# TaskManager

## Summary:

This website allows users add and remove (persistent) tasks  from their schedule

- **Link to project:** https://jarrell-w.github.io/taskmanager/

## How It's Made:

This web app uses DOM manipulation to grab input from the user and upon submit, updates the website to add their task. It users local browser storage to save user tasks.

- **Tech used:** HTML, CSS, JavaScript, Browser API
